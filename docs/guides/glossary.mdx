# Glossary

APR: Annual Percentage Return.

Collection: A set of NFTs within the same contract.

Collection Offer: An offer which gets submitted on all items within a collection.

Contract: An Motoko smart contract

Floor Price: The lowest amount of money required to spend to obtain an NFT using the Buy Now function.

Fee: The cost required to perform a transaction, priced in ICP.

Memo: Memo attached to transfer.

NDP: NDP is the governance token of NnsDAO Protocol.

subDAO: A split DAO or a subset of DAOs called DAOs.

PFPs: PFPs are profile pictures, but for people in cryptocurrency, it is an identity and a place to show off their wealth, character, & ability to the world.
